# Kiasili Ventures

Premium landing page for Kiasili Ventures, a Ugandan business group operating across hospitality, dining, and bakery services.

## Tech Stack

- Next.js App Router
- TypeScript
- Tailwind CSS

## Local Development

```bash
npm run dev
```

Open `http://localhost:3000` in your browser.

## Production Build

```bash
npm run build
```

The project is prepared for deployment on **Cloudflare Workers**.

## Notes

- The logo asset lives in `public/images/Kiasili Ventures Logo.png`.
- The homepage uses local font stacks and does not depend on remote font loading.
- Metadata, favicon wiring, and organization schema are set up in the app.
